▎Описание приложения

Это приложение предназначено для шифрования текста, введённого пользователем. Оно поддерживает несколько методов шифрования: 

1. Код Цезаря - простой шифр, который сдвигает каждую букву текста на фиксированное количество позиций в алфавите.
  
2. Код ROT1 - это частный случай шифра Цезаря, при котором каждая буква смещается на одну позицию.

3. Двоичный код - перевод символов текста в двоичный формат, что позволяет представлять их в виде нулей и единиц.

▎Установка

Для установки приложения необходимо иметь установленный Python. Вы можете загрузить его с официального сайта Python (https://www.python.org/downloads/).


▎Использование

Для запуска приложения выполните следующее:

1. Запустите скрипт:

python encryptor.py


2. Выберите метод шифрования.
3. Выберете шифровку/расшифровку текста.
4. Введите текст, который вы хотите зашифровать.
5. Получите  результат.
